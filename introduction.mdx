## Welcome to ZeroEval TypeScript SDK

ZeroEval helps you build better AI products with comprehensive observability, evals, and A/B testing. The TypeScript SDK provides automatic tracing for your AI applications with minimal code changes.

## Key features

<CardGroup cols={2}>
  <Card
    title="Span decorator & tracer API"
    icon="wand-magic-sparkles"
  >
    Instrument any function with a single line and capture sessions, traces and spans easily
  </Card>
  <Card
    title="Automatic integrations"
    icon="plug"
  >
    OpenAI client, Vercel AI SDK, and LangChain/LangGraph are traced automatically with optional peer dependencies
  </Card>
  <Card
    title="Works everywhere"
    icon="globe"
  >
    Node 18+, Bun, and browser environments (Vite / Next.js)
  </Card>
  <Card
    title="Comprehensive observability"
    icon="chart-line"
  >
    Track sessions, traces, spans, errors, and custom signals in real-time
  </Card>
</CardGroup>
